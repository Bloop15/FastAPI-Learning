# FastAPI Learning 

This repository contains multiple practice modules created while learning FastAPI.  
Each module demonstrates different FastAPI concepts such as routing, database operations, authentication, middleware, templates, and file uploads.

## How to Run

Install the required dependencies:

```bash
pip install fastapi uvicorn sqlalchemy sqlmodel jinja2 python-multipart passlib[bcrypt] python-jose python-dotenv
```

## To start a module (example: basics/main.py):

```bash
uvicorn basics.main:app --reload
```

## Open the application in a web browser:

```bash
http://127.0.0.1:8000
http://127.0.0.1:8000/docs
```

## Purpose

-Practice FastAPI fundamentals
-Learn backend concepts with python
-Organized topic-wise for easy reference
-Useful for future FastAPI development
